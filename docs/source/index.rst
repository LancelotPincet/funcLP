**funcLP** is a library written by **Lancelot PINCET**.
The source code can be consulted on its `GitHub repository <https://github.com/LancelotPincet/funcLP>`_.
However for simply using the features, you can install it in your Python environment using `pip <https://pypi.org/project/funcLP>`_.
And for information, do not hesitate to consult the `documentation <https://funcLP.readthedocs.io>`_.
